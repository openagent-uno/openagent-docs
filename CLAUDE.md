# CLAUDE.md

## Before making any change

Read OpenAgent's vision document first:
[vision.md](./vision.md) (also published at https://github.com/openagent-uno/openagent-docs/blob/main/vision.md)

It is the source of truth for OpenAgent's long-term goals and project standards. When the implementation and vision.md diverge, vision.md is correct and the implementation is what needs to change. Edits to the rest of this docs site — guide pages, examples, the index — must stay consistent with it.
