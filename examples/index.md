# Examples

Use these files as a starting point for real deployments.

## Included Assets

- [Example `openagent.yaml`](./openagent-yaml.md)
- [Example `workspace-mcp.service`](./workspace-mcp-service.md)

## Notes

- The sample config shows the main sections OpenAgent expects: model, channels, memory, scheduler, and auto-update.
- The service file is useful when you want a long-running MCP-sidecar or workspace-specific systemd user service.
- Copy the example and replace environment variables, tokens, and local paths with your own values.
